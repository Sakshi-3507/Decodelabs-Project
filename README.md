# 📊 Data Analytics Internship Project

## About This Project

This repository contains the work I completed during my Data Analytics Internship. The project follows a complete data analysis workflow, starting from cleaning raw data, performing exploratory data analysis (EDA), creating visualizations, and finally using SQL to extract meaningful business insights.

Working on this project helped me understand how raw data is transformed into valuable information that supports decision-making.

---

# 📌 Task 1 – Data Cleaning

## What I Was Trying to Do

The first step was to clean a raw dataset before performing any analysis. The dataset contained missing values, duplicate records, inconsistent formatting, and unnecessary data that needed to be fixed.

My objectives were:

* Remove duplicate records
* Handle missing values
* Standardize column names
* Clean inconsistent text values
* Convert data into proper formats
* Create a clean dataset ready for analysis

---

## Tools I Used

* Python
* Pandas
* NumPy
* Google Colab
* Microsoft Excel

---

## Cleaning Process

### 1. Importing the Dataset

Loaded the Excel dataset into Pandas for analysis.

### 2. Handling Missing Values

Identified missing values and either filled or removed them depending on the importance of the column.

### 3. Removing Duplicate Records

Checked for duplicate rows and removed them to improve data quality.

### 4. Data Formatting

Converted columns into appropriate formats, including dates and numerical values.

### 5. Data Validation

Performed final checks to ensure the dataset was clean, consistent, and ready for analysis.
